# 🚀 Developer Journey Program

A simple and creative Python program that represents the journey of a developer using **Object-Oriented Programming (OOP)** concepts.

This project displays a developer's profile, goals, programming language preference, and motivation behind learning to code.

## 📌 About The Project

Every developer begins with a single line of code.  
This program represents the idea of learning, building, and improving step by step.

It includes:

- Developer information
- Career goal
- Favorite programming language
- Personal motto
- Developer inspiration

✅ Class and Object  
✅ Constructor (`__init__`)
